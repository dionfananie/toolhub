## 5. SEO Spec per Artikel

| Elemen | Target |
|---|---|
| Title | < 60 karakter, keyword di depan |
| Meta description | 120–155 karakter, ada keyword |
| URL slug | Bahasa Indonesia, hyphen-separated, < 80 karakter |
| Heading | 1× `<h1>`, bertingkat `h2` → `h3` |
| Word count | 800–2.000 kata |
| Internal links | 3–5 ke halaman tool dan blog post lain |
| External links | 2–3 ke sumber otoritatif (MDN, web.dev, docs resmi) |
| Gambar | Minimal 1 custom image (OG 1200×630) |
| Structured data | `Article` + `BreadcrumbList` JSON-LD |
